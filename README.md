# Rofi Metro
Windows 8-like theme for Rofi


## What's inside

### Start Screen
A Rofi theme that mimicks Windows 8's Start screen UI.

### Charms
A Charms bar-like Rofi theme with clickable and configurable buttons to launch programs.
